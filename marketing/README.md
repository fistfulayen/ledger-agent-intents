## What We Built

| Feature | Description |
|---------|-------------|
| **Intent Queue** | Agents submit payment requests; users review and approve |
| **Ledger DMK Integration** | Direct device connection (USB/Bluetooth) |
| **Agent Key Provisioning** | LKRP-based cryptographic credentials |
| **x402 Protocol** | Pay-per-call API payments (EIP-3009) |
| **Status Lifecycle** | pending → approved → authorized → confirmed |

### Supported
- **Chains:** Base, Base Sepolia, Sepolia
- **Token:** USDC
- **Signing:** EIP-712, EIP-3009, EIP-191
